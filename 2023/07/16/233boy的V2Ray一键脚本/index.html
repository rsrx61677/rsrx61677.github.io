233boy的V2Ray一键脚本

发表于 2023-07-16 更新于 2023-07-20 分类于 网络

前言

-写这篇文章的目的是因为本人比较健忘，所以做个记录以便日后再折腾的时候进行查阅

-本人文化水平有限，如果有语法、错字、标点错误请忽略，我自己能看懂就好

安装

  • 系统支持：Ubuntu，Debian，CentOS，推荐使用 Ubuntu 22，谨慎使用 CentOS，脚本可能无法正常运行！

  • 执行如下命令：bash <(wget -qO- -o- https://git.io/v2ray.sh)

管理面板

  • 安装完成后，输入 v2ray 就能看到管理面板

帮助

  • 使用：v2ray help

目录

  • V2Ray 脚本全部保存在 /etc/v2ray

  • 脚本：/etc/v2ray/sh

  • 核心：/etc/v2ray/bin

  • 配置：/etc/v2ray/conf
