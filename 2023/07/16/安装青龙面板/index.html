安装青龙面板

发表于 2023-07-16 更新于 2023-07-20 分类于 网络

前言

-写这篇文章的目的是因为本人比较健忘，所以做个记录以便日后再折腾的时候进行查阅

-本人文化水平有限，如果有语法、错字、标点错误请忽略，我自己能看懂就好

环境

  • 安装Docker

    使用官方安装脚本自动安装命令如下：

    curl -fsSL https://get.docker.com -o get-docker.sh

    sudo sh get-docker.sh

特征

  • 支持多种脚本语言（python3，javaScript，shell，typescript）

  • 支持脚本、环境变量、配置文件的在线管理

  • 支持在线查看任务日志

  • 支持秒级任务设置

  • 支持系统级通知

  • 支持夜间模式

  • 支持手机操作

安装

使用

  • 更新并重启青龙

    ql update

  • 运行自定义脚本

    ql extra

  • 添加单个脚本文件

    ql raw <脚本地址>

  • 添加某个库的单个脚本

    ql repo <脚本地址> <白名单> <黑名单> <依赖> <分支>

  • 删除旧日志

    ql rmlog <日志保留天数>

  • 启动机器人

    ql bot

  • 检测青龙环境并修复

    ql check

  • 重置登录错误次数

    ql resetlet

  • 禁用两步验证

    ql resettfa
